Coreseek的安装测试

  • coreseek
  • linux
  • mysql
  • php
  • sphinx
linuxmysqlphpsphinx

Coreseek安装文档：http://www.coreseek.cn/products-install/

1.设置路径和中文环境

export PATH=/usr/local/bin:$PATH
export LC_ALL=zh_CN.UTF-8
export LANG=zh_CN.UTF-8
**2.下载Coreseek** 
weget http://www.coreseek.cn/uploads/csft/3.2/coreseek-3.2.14.tar.gz
tar xzvf coreseek-3.2.14.tar.gz
cd coreseek-3.2.14

3.安装mmseg

cd mmseg-3.2.14
./bootstrap
./configure --prefix=/usr/local/mmseg3
make && make install

如果是Mac OS X:

wget https://raw.githubusercontent.com/Homebrew/homebrew/9c7802270aabf638654ea0bccecc592e04785f87/Library/Formula/mmseg.rb -O/usr/local/Library/Formula/mmseg.rb
brew install mmseg

4.安装coreseek

cd csft-3.2.14
sh buildconf.sh
./configure --prefix=/usr/local/coreseek  --without-unixodbc --with-mmseg --with-mmseg-includes=/usr/local/mmseg3/include/mmseg/ --with-mmseg-libs=/usr/local/mmseg3/lib/ --with-mysql
make && make install

4.测试mmseg分词，coreseek搜索

cd testpack
/usr/local/mmseg3/bin/mmseg -d /usr/local/mmseg3/etc var/test/test.xml
/usr/local/coreseek/bin/indexer -c etc/csft.conf --all
/usr/local/coreseek/bin/search -c etc/csft.conf

测试效果可以参考：http://www.coreseek.cn/products/products-install/install_on_bsd_linux/第三点
