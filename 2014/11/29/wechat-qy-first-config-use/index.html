微信企业号回调模式如何开启及使用

  • php
  • 微信
php

1.首先你得在微信企业号后台开启某个应用的回调模式


 

2.回调模式的使用（即代码的实现）

首先下载微信的PHP SDK，下载地址：https://github.com/dodgepudding/wechat-php-sdk

在你开启回调模式是填写的URL地址（如：http://www.jsyzchen.com/index.php）里写下如下代码:


<?php
include “./wechat/qywechat.class.php”;//引入企业微信号的类
function logdebug($text){ //调试输出方法
    file_put_contents(‘./data/log.txt’,$text.”\n”,FILEAPPEND);
};
$options = array(
    ‘token’=>’qyweixin’, //填写应用接口的Token
    ‘encodingaeskey’=>’’,//填写加密用的EncodingAESKey
    ‘appid’=>’’, //填写高级调用功能的appid,及企业号ID
    ‘appsecret’=>’’, //填写高级调用功能的密钥
    //‘datapath’=>’./data/cookie‘,
    ‘debug’=>true,
    ‘logcallback’=>’logdebug’, //调试输出方法，需要有一个string类型的参数
    ‘agentid’=>’2’, //应用的id
);
$weObj = new Wechat($options);//实例化对象
$res = $weObj->valid(); //注意, 企业号与普通公众号不同，必须打开验证，不要注释掉
if (!$res) {
    logdebug(“验证失败！”);
    var_dump($res);
    exit;
}
$type = $weObj->getRev()->getRevType();//得到收到信息的类型
switch($type) {
    case Wechat::MSGTYPE_TEXT://文本类型
        $weObj->text(“hello, I’m text”)->reply();
        break;
    case Wechat::MSGTYPE_EVENT://事件类型
        $weObj->text(“hello, I’m event”)->reply();
        break;
    case Wechat::MSGTYPE_IMAGE://图片类型
        $weObj->text(“已收到图片”)->reply();
        break;
    default:
        $weObj->text(“help info”)->reply();
}

好了，这样你就可以用微信客户端发送消息给企业号应用喽，它可以自动回复哦。。
